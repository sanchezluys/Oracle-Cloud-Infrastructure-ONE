### Crear Cuenta OCI Free Tier

```mermaid
flowchart TD
    %% Version 1.0: sanchezluys 2024
    inicio((**Inicio**))-->
    crearCuenta[Crear Cuenta Free Tier 30 dias con crédito] 
    --> accederOracle[**Acceder a Oracle Cloud**]
    click accederOracle "https://www.oracle.com/cloud/free/" "Ir a Oracle Cloud Free Tier"
    accederOracle --> registrarDatos[Se registra con datos básicos: _pais_, _nombre_, _apellido_, _correo_]
    registrarDatos --> verificarCorreo[El correo debe ser verificado]
    verificarCorreo --> correoLlego{¿El correo de validación llegó?}
    correoLlego -->|**SI** | validarCuenta[**Validar Cuenta**: Clic en el enlace de validación]
    correoLlego -->|**NO** | reintentarRegistro[**Reintentar Registro**]
    reintentarRegistro --> accederOracle
    validarCuenta --> seguirRegistro[Seguir al 2do registro, _clave_, _nombre-empresa_, **region-principal**, _aceptar-condiciones_]
    seguirRegistro --> formDatosPer[Formulario Direccion]
    formDatosPer --> formTDC[Formulario Tarjeta de Crédito]
    formTDC --> aceptaTDC{Tarjeta Aceptada}
    aceptaTDC --> |SI| iniciarPrueba[Iniciar Prueba]
    aceptaTDC --> |NO| reintentarTDC[Reintentar]-->formTDC
    iniciarPrueba --> dobleFactor[Activación doble factor con apk de Oracle]
    -->cuentaActiva[Cuenta creada con éxito]
    -->finDiagrama((**Fin**))
    %%********************************
    %% Definir una clase para el estilo
    classDef inicio fill:green,stroke:#333,stroke-width:2px, color:#000
    classDef bloque fill:cyan,stroke:#333,stroke-width:2px, color:#000
    classDef condicional fill:yellow,stroke:#333,stroke-width:2px, color:#000
    classDef fin fill:brown,stroke:#333,stroke-width:2px, color:#000
    %%********************************
    %% Aplicar el estilo a varios bloques
    class inicio inicio
    class accederOracle,registrarDatos,verificarCorreo,reintentarRegistro,validarCuenta,reintentarRegistro,seguirRegistro,formDatosPer,formTDC bloque
    class iniciarPrueba,reintentarTDC,cuentaActiva,dobleFactor,crearCuenta bloque
    class correoLlego,aceptaTDC condicional
    class finDiagrama fin
```

### Comandos desde la consola o terminal de OCI

| Item | Comando | Uso |
| --------- | --------- | --------- |
|     1      | uname -m          |  muestra el SO, por ejemplo: aarch64, linux         |
|   2        |   oci --version        |  muestra la versión de OCI  ejm: 3.49.2       |
|   3        |  oci iam user list         |           |
